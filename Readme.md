## 1. Objective
This project is a **console-based Library Management System** developed using **Python Object-Oriented Programming (OOP)** concepts.  
It allows users to manage **books**, **borrowers**, and perform operations like **borrowing, returning**, and **searching books**.  
The system uses **in-memory storage** (no database), making it simple and suitable for learning core OOP concepts.
